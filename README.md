# Sivers SIDIS Fit Reproduction

This repository contains a Python implementation using the **iminuit** package
to reproduce the Sivers asymmetry fit results presented in:

[Boglione et al., JHEP 07 (2018) 148](https://inspirehep.net/literature/1680033)

* Currently, the fitted central values are fairly well matched (although not exactly) with the published results.
